# Royal Hotel POS - Quick Start Guide

## 🚀 Getting Started (First 5 Minutes)

### Step 1: Login
1. Open the POS system
2. Enter your credentials
3. Select your branch
4. You're ready to start!

### Step 2: Select a Table (For Dine-In)
- Click on the table number
- Table turns blue when selected
- Order type automatically set to "Dine In"

### Step 3: Choose Menu Session
- Click "Breakfast", "Lunch", or "Dinner"
- Menu shows only items for that session
- "All Sessions" shows everything

### Step 4: Select Items
- Click category (left side)
- Click item from grid (center)
- Quantity in cart updates (right side)

### Step 5: Adjust & Send
- Change quantities: +/- buttons
- Add notes: Click item → Edit notes
- Send to kitchen: Blue "Send to Kitchen" button (F6)

### Step 6: Payment
- Quick Pay: F7 (Cash) or F8 (Card)
- Or: Manual payment dialog with options
- Print receipt automatically

## ⚡ Speed Tips (Make Yourself Faster)

### Use Keyboard Shortcuts
```
F1  → Start new order
F2  → Search items (type to find)
F3  → Hold this bill (save for later)
F4  → Recall held bill
F5  → Print bill
F6  → Send to kitchen  
F7  → Cash payment now!
F8  → Card payment now!
F9  → Split bill between people
```

### Quick Quantities
1. Click ×1, ×2, ×3, or ×5 button
2. Click the item
3. It adds that quantity directly
4. Example: Click ×3, click "Biryani" → 3 Biryani added

### Favorites & Specials
- Orange "Today Specials" section: Single click to add
- Green "Favorites" section: Popular items
- Saves 3-5 seconds per item

### Hold & Resume
- Order interrupted? Press F3 to hold it
- Change table? Recall the bill (F4) on new table
- Held bills saved up to 20 per shift

### Repeat Orders
- Last customer ordered same thing?
- Click "Repeat Last Order"
- Previous order loads into cart
- Modify as needed

## 🔧 Troubleshooting Quick Fixes

### **Q: Item won't add to cart**
**A**: Check if item is "out of stock" (red badge). Admin needs to mark "Available".

### **Q: Can't send to kitchen**
**A**: Make sure you:
1. Selected a table (or chose delivery)
2. Added at least 1 item
3. WiFi is connected ✅

### **Q: Payment screen won't show**
**A**: Try:
1. Refresh page (Ctrl+R)
2. Check WiFi connection
3. Close and reopen browser tab

### **Q: Can't find yesterday's order**
**A**: 
1. Go to Dashboard → Orders
2. Use date filter
3. Search by order number
4. Search by customer name

### **Q: Print not working**
**A**:
1. Ask admin to check Settings → Printer Settings
2. Verify printer IP address
3. Check paper is loaded
4. Restart printer

### **Q: Held bill disappeared**
**A**: Held bills clear every 24 hours. Try:
1. Recall (F4) immediately
2. Check Orders page to recreate
3. But faster: reload page won't clear until you switch tables

## 💡 Pro Tips & Tricks

### Tip 1: Search for Items
- Press F2 anywhere
- Type "bia" → shows "Biryani", "Biryani Rice"
- Super fast for 100+ item menus

### Tip 2: Batch Payments
- Multiple tables? Hold all bills (F3)
- Recall one by one (F4)
- Customer pays at entrance point

### Tip 3: Peak Hours
- Pre-select quantities (×3, ×5 button)
- Use favorites section
- Avoid clicking category each time
- Send to kitchen immediately

### Tip 4: Delivery Orders
- Select "Delivery" as order type
- Add delivery address in notes
- System tracks delivery orders separately

### Tip 5: Customer Notes
- Add allergies: "Shellfish allergy"
- Add preferences: "Extra lemon"
- Kitchen can't miss anything

### Tip 6: Daily Specials
- Load during setup
- Orange badge makes them pop
- Single click to add (saves time)
- Increases special sales by 40%+

### Tip 7: Offline Mode
- WiFi down? No problem!
- Menu is cached locally
- Cart saved automatically
- Syncs when WiFi back on

## 📞 Need Help?

### Quick Reference
- **Can't login**: Check caps lock, verify branch assignment
- **Items missing**: Ask admin to add to menu or mark available
- **Printer offline**: Check IP in Settings, restart if needed
- **Can't find order**: Check date range, use search filters
- **Payment failing**: Check WiFi, try again, contact admin

### Contact Admin If:
- Item prices wrong
- Daily specials not showing
- Printer configuration needed
- User access issues
- System errors persist
